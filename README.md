# docker Oxid6xMollie
 Docker container with OXID and Mollie


1. Install Docker

2. Install Oxid/Mollie

git clone https://github.com/SalimAtMollie/docker-Oxid6xMollie MollieOxid

cd MollieOxid

docker-compose build

docker-compose up
